## Aim
To perform a heuristic evaluation of two leading e-commerce platforms, **Amazon** and **Flipkart**, based on major usability principles, compare their UI/UX, identify usability issues, and provide improvement suggestions.

---

## Algorithm
1. Select Amazon and Flipkart as the two competing websites.
2. Analyze both websites based on 6 major heuristic categories.
3. Capture screenshots for each heuristic category.
4. Compare usability, interface clarity, and navigation flow.
5. Document observations for each heuristic.
6. Conclude with recommendations and improvements.

## Result
The evaluation concludes that:

- **Amazon** excels in search accuracy, product detail clarity, checkout flow, and accessibility.  
- **Flipkart** excels in navigation simplicity, homepage design, and category browsing.  

Both platforms can improve:
- Reducing homepage clutter  
- Enhancing filter options  
- Improving error messages and consistency  

This evaluation highlights important UX insights and design improvements for better user experience.
